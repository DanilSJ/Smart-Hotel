# Smart Hotel API

![Smart Hotel](https://img.shields.io/badge/Project-Smart%20Hotel-blue)

## Описание проекта

**Smart Hotel API** — это серверная часть для умного отеля, где большинство процессов автоматизировано с помощью роботов. Проект реализует управление номерами, бронирование, доставку еды и вещей роботами, контроль лифтов и отслеживание статуса сервисов в реальном времени.  

Проект предназначен для демонстрации возможностей современных технологий в гостиничном бизнесе, а также для включения в портфолио разработчика.

---

## Технологии

- **Python 3.11+** — основной язык программирования  
- **FastAPI** — веб-фреймворк для создания REST API  
- **SQLAlchemy** — ORM для работы с базой данных  
- **Uvicorn** — ASGI сервер для запуска FastAPI  
- **Pydantic** — для валидации данных  

---

## Основные функциональные возможности

1. **Пользователи и авторизация**  
   - Регистрация и вход гостей и сотрудников  
   - Управление профилем пользователя  

2. **Номера и бронирование**  
   - Просмотр доступных номеров  
   - Бронирование и управление статусом номера  

3. **Роботы и доставка**  
   - Управление роботами  
   - Назначение доставок еды и вещей  
   - Отслеживание статуса доставки в реальном времени  

4. **Лифты**  
   - Вызов и управление лифтами  
   - Проверка текущего местоположения  

5. **Ресепшн и сервис**  
   - Заезд и выезд гостей  
   - Создание и управление запросами гостей  

6. **Уборка и обслуживание**  
   - Управление задачами по уборке  
   - Отслеживание выполнения задач  

7. **Меню и заказы**  
   - Просмотр меню  
   - Создание и отслеживание заказов еды

## Основные эндпоинты

### Пользователи
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST  | `/auth/register` | Регистрация |
| POST  | `/auth/login`    | Вход |
| POST  | `/auth/logout`   | Выход |
| GET   | `/users/{user_id}` | Информация о пользователе |
| PATCH | `/users/{user_id}` | Редактирование профиля |

### Номера
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET   | `/rooms` | Список всех номеров |
| GET   | `/rooms/{room_id}` | Информация о номере |
| PUT   | `/rooms/book` | Забронировать номер |
| GET   | `/rooms/{room_id}/availability` | Проверить доступность |

### Роботы и доставка
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET   | `/robots` | Список всех роботов |
| GET   | `/robots/{robot_id}` | Информация о роботе |
| PATCH | `/robots/{robot_id}/status` | Обновить статус робота |
| POST  | `/robots/{robot_id}/deliver` | Назначить доставку |
| GET   | `/deliveries` | Все доставки |
| GET   | `/deliveries/{delivery_id}` | Статус доставки |
| PATCH | `/deliveries/{delivery_id}/complete` | Завершить доставку |

### Лифты
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET   | `/elevators` | Список лифтов |
| POST  | `/elevators/{elevator_id}/call` | Вызвать лифт |
| POST  | `/elevators/{elevator_id}/move` | Переместить лифт |
| GET   | `/elevators/{elevator_id}/status` | Статус лифта |

### Ресепшн и сервис
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST  | `/reception/checkin` | Заезд гостя |
| POST  | `/reception/checkout` | Выезд гостя |
| GET   | `/reception/requests` | Текущие запросы |
| POST  | `/reception/request` | Новый запрос |
| PATCH | `/reception/request/{request_id}` | Обновление статуса запроса |

### Меню и заказы
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET   | `/menu` | Список блюд |
| GET   | `/menu/{dish_id}` | Информация о блюде |
| POST  | `/orders` | Создать заказ еды |
| GET   | `/orders` | Текущие заказы |
| PATCH | `/orders/{order_id}/status` | Обновить статус заказа |

### Отслеживание в реальном времени
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET   | `/status/robot/{robot_id}` | Текущая задача робота |
| GET   | `/status/delivery/{delivery_id}` | Статус доставки |
| GET   | `/status/elevator/{elevator_id}` | Местоположение лифта |
